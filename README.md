# Nixie-clock-ESP32
My new nixie clock  project based on esp32 lolin lite and LC-516 nixie tubes

Main componnents:
-ESP32 lolin lite 
-Nixie lamps LC-516
-Nixie drivers 74141
-Scift registers 74HC595
-HV voltage converter
